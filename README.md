Пояснительная записка
К проекту: "Wyvern: The Path to the Crown of Heaven"
Авторы: Артемий Козлов, Игорь Семенков
1. Цель документа
Эта записка дополняет описание игры и материалы презентации. В ней объясняются основные идеи, почему мы выбрали минимум текста, и даны советы по показу. Всё основано на готовом коде и пожеланиях пользователя.
2. Почему этот проект
Игра — это 2D приключение в красивом пиксельном стиле (сделано на Python). Вдохновлено классическими фэнтези: рыцарь сражается с силами тьмы за "Корону небес". Геймплей простой и увлекательный: плавные анимации, динамичные бои, удобный интерфейс. В презентации акцент на видео и картинках — чтобы было ярко и запоминающеся.
3. Краткое описание игры
Жанр: 2D платформер с боями.
Сюжет: Короткий пролог с субтитрами: сын рыцаря побеждает тьму в замке. Уровни: уютная деревня → тёмное подземелье.
Как играть:

Герой может атаковать, уклоняться, у него 100 единиц здоровья.
Враги — скелеты и подобные.
Интерфейс: полоска здоровья, меню паузы, экран смерти.
Настройки: звук, сохранения в файл.

Техническая основа: Готовые карты уровней, плавная камера, реалистичная физика движений.
